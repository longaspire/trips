TRIPS

The rapid accumulation of indoor positioning data is increasingly booming the interest in indoor mobility analyses. As a fundamental analysis, it is highly relevant to translate raw indoor positioning data into mobility semantics that describe what, where and when in a more concise and semantics-oriented way. Such a translation is challenging as multiple data sources are involved, the raw indoor positioning data is of low quality, and the translation results are hard to assess. In this work, we demonstrate a system TRIPS that streamlines the entire translation process by three functional components. The Configurator provides a standard but concise means to configure multiple input sources, including the indoor positioning data, indoor space information, and relevant context. The Translator cleans the indoor positioning data and exports reliable mobility semantics without manual interventions. The Viewer offers a suite of flexible operations to trace the input, output and intermediate data involved in the translation. Data analysts can interact with TRIPS to obtain their desired mobility semantics in a visual and convenient way.

Features

We design a system named TRIPS. It provides an end-to-end solution to translate the raw indoor positioning data into the visual mobility semantics in support of the semanticsoriented indoor mobility analysis. In general, TRIPS is characterized by several desirable features:

  • It provides a concise, labor-saving manner to set up positioning data source and manage contextual resources, making it highly convenient to arrange the input data sources.
  • It adopts a three-layer translation framework to deal with the inherent discreteness and uncertainty in the raw indoor positioning data. The techniques in the cleaning layer, annotation layer and complementing layer help export reliable mobility semantics jointly.
  • It offers a suite of flexible interactions to visually trace the input, output, and intermediate data at the same time, which is very intuitive for users to verify the translation results.
